# Customer Behavior Analysis Dashboard

## Overview

This project analyzes customer shopping behavior using Python, MySQL, and Power BI. The objective is to identify purchasing patterns, customer segments, and sales trends to support data-driven business decisions.

## Tools & Technologies

* Python (Pandas)
* MySQL
* Power BI

## Project Workflow

1. Data Cleaning & Preprocessing using Python
2. Exploratory Data Analysis (EDA)
3. Business Analysis using SQL Queries
4. Dashboard Development in Power BI
5. Insight Generation and Reporting

## Key Features

* Analyzed shopping behavior data of 3.9K customers.
* Performed data cleaning and exploratory data analysis (EDA).
* Used SQL to analyze customer demographics, purchase patterns, and category performance.
* Built an interactive Power BI dashboard with key business KPIs.
* Created customer segmentation based on age group, gender, subscription status, and shipping preferences.

## Dashboard KPIs

* Total Customers: 3.9K
* Average Purchase Amount: $59.76
* Average Review Rating: 3.75

## Dashboard Insights

* Revenue by Product Category
* Sales by Category
* Revenue by Age Group
* Customer Segmentation Analysis
* Subscription Status Distribution
* Shipping Preference Analysis

The analysis helps identify high-value customer segments, understand purchasing behavior, evaluate product category performance, and support data-driven decision-making.
